## get_group_info

Get group information.

### Request

<ParamField path="group_id" type="string" required>
  Group number
</ParamField>

### Response

<ResponseField name="group_id" type="number">
  Group number
</ResponseField>

<ResponseField name="group_name" type="string">
  Group name
</ResponseField>

<ResponseField name="group_remark" type="string">
  Group remark (note)
</ResponseField>

<ResponseField name="group_all_shut" type="number">
  All members muted: `-1` if enabled, `0` if disabled
</ResponseField>

<ResponseField name="member_count" type="number">
  Current member count
</ResponseField>

<ResponseField name="max_member_count" type="number">
  Maximum member capacity
</ResponseField>

### Example

```json
// Request
{
  "group_id": "123456"
}

// Response
{
  "status": "ok",
  "retcode": 0,
  "data": {
    "group_id": 123456,
    "group_name": "Example Group",
    "group_remark": "My Group",
    "group_all_shut": 0,
    "member_count": 50,
    "max_member_count": 200
  }
}
```

## get_group_list

Get the list of groups the bot has joined.

### Request

No parameters required.

### Response

<ResponseField name="data" type="array">
  Array of group objects
</ResponseField>

Each group object contains:
- `group_id` (number): Group number
- `group_name` (string): Group name
- `group_remark` (string): Group remark
- `group_all_shut` (number): Mute status
- `member_count` (number): Member count
- `max_member_count` (number): Max capacity

### Example

```json
// Response
{
  "status": "ok",
  "retcode": 0,
  "data": [
    {
      "group_id": 123456,
      "group_name": "Group 1",
      "group_remark": "My Group 1",
      "group_all_shut": 0,
      "member_count": 50,
      "max_member_count": 200
    },
    {
      "group_id": 789012,
      "group_name": "Group 2",
      "group_remark": "My Group 2",
      "group_all_shut": -1,
      "member_count": 100,
      "max_member_count": 500
    }
  ]
}
```

## get_group_member_info

Get information about a group member.

### Request

<ParamField path="group_id" type="string" required>
  Group number
</ParamField>

<ParamField path="user_id" type="string" required>
  Member QQ number
</ParamField>

<ParamField path="no_cache" type="boolean | string" default="false">
  Bypass cache and fetch fresh data
</ParamField>

### Response

<ResponseField name="group_id" type="number">
  Group number
</ResponseField>

<ResponseField name="user_id" type="number">
  Member QQ number
</ResponseField>

<ResponseField name="nickname" type="string">
  Member nickname
</ResponseField>

<ResponseField name="card" type="string">
  Group card (display name in group)
</ResponseField>

<ResponseField name="sex" type="string">
  Gender: `male`, `female`, or `unknown`
</ResponseField>

<ResponseField name="age" type="number">
  Age
</ResponseField>

<ResponseField name="join_time" type="number">
  Join time (Unix timestamp)
</ResponseField>

<ResponseField name="last_sent_time" type="number">
  Last message time (Unix timestamp)
</ResponseField>

<ResponseField name="level" type="string">
  Group level
</ResponseField>

<ResponseField name="qq_level" type="number">
  QQ level
</ResponseField>

<ResponseField name="role" type="string">
  Role: `owner`, `admin`, or `member`
</ResponseField>

<ResponseField name="title" type="string">
  Special title
</ResponseField>

<ResponseField name="area" type="string">
  Location/area
</ResponseField>

<ResponseField name="unfriendly" type="boolean">
  Has bad record
</ResponseField>

<ResponseField name="title_expire_time" type="number">
  Special title expiration time
</ResponseField>

<ResponseField name="card_changeable" type="boolean">
  Can change group card
</ResponseField>

<ResponseField name="shut_up_timestamp" type="number">
  Mute expiration timestamp
</ResponseField>

<ResponseField name="is_robot" type="boolean">
  Is a robot/bot account
</ResponseField>

### Example

```json
// Request
{
  "group_id": "123456",
  "user_id": "987654"
}

// Response
{
  "status": "ok",
  "retcode": 0,
  "data": {
    "group_id": 123456,
    "user_id": 987654,
    "nickname": "User",
    "card": "Card Name",
    "sex": "male",
    "age": 20,
    "join_time": 1234567890,
    "last_sent_time": 1234567890,
    "level": "10",
    "qq_level": 64,
    "role": "member",
    "title": "Special Member",
    "shut_up_timestamp": 0,
    "card_changeable": true,
    "is_robot": false
  }
}
```

## set_group_ban

Mute a group member.

### Request

</ParamField>

  Member QQ number to mute
</ParamField>

  Mute duration in seconds (0 to unmute, max 2592000 = 30 days)
</ParamField>

<ResponseField name="result" type="null">
  Returns `null` on success
</ResponseField>

### Example

```json
{
  "group_id": "123456",
  "user_id": "987654",
  "duration": 3600
}

// Request - Unmute
{
  "group_id": "123456",
  "user_id": "987654",
  "duration": 0
}

// Response
{
  "status": "ok",
  "retcode": 0,
  "data": null
}
```

## set_group_add_request

Approve or reject a group join request.

### Request

<ParamField path="flag" type="string" required>
  Request flag from group request event
</ParamField>

<ParamField path="sub_type" type="string" required>
  Request type: `add` (join request) or `invite` (invitation)
</ParamField>

<ParamField path="approve" type="boolean | string" default="true">
  Approve (`true`) or reject (`false`)
</ParamField>

<ParamField path="reason" type="string" default="">
  Rejection reason (only used when rejecting)
</ParamField>

### Response

<ResponseField name="result" type="null">
  Returns `null` on success
</ResponseField>

### Example

```json
// Request - Approve
{
  "flag": "request_flag_12345",
  "sub_type": "add",
  "approve": true
}

// Request - Reject
{
  "flag": "request_flag_12345",
  "sub_type": "add",
  "approve": false,
  "reason": "Not accepting new members"
}

// Response
{
  "status": "ok",
  "retcode": 0,
  "data": null
}
```
