## Overview

NapCat provides extended actions beyond the standard OneBot 11 protocol. These actions enable advanced features specific to NapCat and the NTQQ platform.

## Image & Media

### ocr_image

Perform OCR (text recognition) on an image.

<Warning>
  This action is only supported on Windows platform.
</Warning>

#### Request

<ParamField path="image" type="string" required>
  Image path, URL, or Base64-encoded image data
</ParamField>

#### Response

<ResponseField name="texts" type="array">
  Array of recognized text objects with position and content
</ResponseField>

<ResponseField name="language" type="string">
  Detected language
</ResponseField>

#### Example

```json
// Request
{
  "image": "https://example.com/image.png"
}

// Response
{
  "status": "ok",
  "retcode": 0,
  "data": {
    "texts": [
      {
        "text": "Hello World",
        "confidence": 0.95,
        "coordinates": [[10, 20], [100, 20], [100, 40], [10, 40]]
      }
    ],
    "language": "en"
  }
}
```

## Status & Profile

### set_online_status

Set the bot's online status.

#### Request

<ParamField path="status" type="number | string" required>
  Status code (see status list below)
</ParamField>

<ParamField path="ext_status" type="number | string" required>
  Extended status code
</ParamField>

<ParamField path="battery_status" type="number | string" required>
  Battery status (0-100)
</ParamField>

#### Status Codes

**Basic Status:**
- `10` - Online
- `30` - Away
- `40` - Invisible
- `50` - Busy
- `60` - Q Me
- `70` - Do Not Disturb

**Extended Status (ext_status):**
- `1028` - Listening to Music
- `1018` - Studying
- `1016` - Sleeping
- `1027` - Gaming (TIMI)
- `1300` - Slacking Off
- `1032` - Staying Up Late
- `1401` - Emo
- And many more...

#### Example

```json
// Request - Set to "Busy"
{
  "status": 50,
  "ext_status": 0,
  "battery_status": 100
}

// Request - Set to "Listening to Music"
{
  "status": 10,
  "ext_status": 1028,
  "battery_status": 85
}

// Response
{
  "status": "ok",
  "retcode": 0,
  "data": null
}
```
